# Keyboard shortcuts

A quick reference. See [Keyboard and accessibility](../how-to/keyboard-and-accessibility.md) for the full flow.

## Global

| Keys | Action |
|---|---|
| `Cmd/Ctrl + K` | Open the search palette |
| `↑` / `↓` then `Enter` | Move through search results and open one |
| `Esc` | Close the palette / cancel the current edit or pick-up |

## Board & sidebar

| Keys | Action |
|---|---|
| `Enter` or `Space` (card focused) | Open the focused task |
| `Space` (card or project focused) | Pick up the item for keyboard reordering |
| `↑` `↓` `←` `→` (after pick-up) | Move the item (within/across columns, or up/down the project list) |
| `Space` again | Drop it in the new position |
| `Esc` | Cancel the reorder |

## File editor

| Keys | Action |
|---|---|
| `Cmd/Ctrl + S` | Save the file |
| `Esc` | Cancel editing |
