# Using FlowBoard

The user guide: how to install, set up, and operate FlowBoard day to day. These pages are written for **people using the dashboard**, not for agents (agent-facing rules live in `docs/project-mode/`) and not for contributors (architecture lives in `docs/concepts/` and `docs/adr/`).

## Start here

- [Getting started](getting-started.md) — from a running dashboard to your first project and first task.

## How-to guides

Task-oriented answers to "how do I …?":

**Boards & tasks**

- [Work the Kanban board](how-to/work-the-kanban.md) — columns, drag, sort modes, subtasks, review, trash.
- [Search and filter tasks](how-to/search-and-filter.md) — the `Cmd/Ctrl+K` palette and its query operators.
- [Customize the project overview](how-to/customize-overview.md) — edit the widget grid, presets, GitHub binding.
- [Brainstorm on the canvas and promote to tasks](how-to/canvas-and-promote.md) — notes, clusters, the Specify stepper.

**Projects & files**

- [Manage projects](how-to/manage-projects.md) — create, rename, reorder, and the difference between **archiving** and **deleting**.
- [Browse and edit project files](how-to/browse-and-edit-files.md) — the explorer, the editor, show-hidden, editable paths.

**Platform**

- [Use FlowBoard on a phone or in Telegram](how-to/work-on-mobile.md) — touch drag-and-drop, full-screen sheets, the Mini App.
- [Keyboard and accessibility](how-to/keyboard-and-accessibility.md) — keyboard reorder, focus, screen-reader support.
- [Update FlowBoard](how-to/update-flowboard.md) — apply a new version from the dashboard or CLI.
- [Troubleshooting](how-to/troubleshooting.md) — hook not registered, context not ready, remote auth.
